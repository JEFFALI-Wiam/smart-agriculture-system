# Smart Agriculture System 🌱

IoT project for smart agriculture using ESP8266.

## Features
- Temperature and humidity monitoring
- Soil moisture detection
- Automatic irrigation system
- Automatic fan control

## Components
- ESP8266 NodeMCU
- DHT22 Sensor
- Soil Moisture Sensor
- Relay Module
- Water Pump
- Fan

## Technologies
- Arduino IDE
- Embedded C++
